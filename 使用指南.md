# DNAPL 图像分析流水线 — 使用指南 (v3.0)

## 快速开始

### 第一步：环境准备

**MATLAB 版本要求**: R2018b 或更高版本（推荐 R2020a+）

**必需工具箱**:

- Image Processing Toolbox
- Statistics and Machine Learning Toolbox

**可选工具箱**:

- Curve Fitting Toolbox（溶解模型拟合，无此工具箱时自动使用对数线性回退方案）

检查工具箱是否已安装：

```matlab
ver('images')          % Image Processing Toolbox
ver('stats')           % Statistics and Machine Learning Toolbox
ver('curvefit')        % Curve Fitting Toolbox
```

### 第二步：准备图像

流水线支持两种输入模式：

**模式 1 — 多页 TIF（默认）**: 一个 .tif 文件包含 N 页（N 个时间帧）

默认数据路径：

```
01 MATLAB CODE/01 Microfluid/01 databased/01 Pretest 5/酒精清洗.tif  (72 页, ~1 GB)
```

**模式 2 — 独立文件**: 多个单独的 .tif 或 .png 文件

**图像要求**:

- 格式：TIF（8-bit 或 16-bit 均支持，16-bit 自动转 8-bit）或 PNG
- 通道：RGB 三通道彩色图像（灰度图自动复制为三通道）
- 内容：DNAPL 以粉红色/红色染料标记

### 第三步：配置参数

打开 `config_pipeline.m`。**所有需要修改的参数已经集中在文件最上方的 `USER SETTINGS` 区域（第 10-46 行）**，下方的高级设置通常不需要改动。

#### USER SETTINGS 区域（第 10-46 行）— 必看

| 行号 | 参数 | 你需要做什么 | 说明 |
|------|------|-------------|------|
| **18-19** | `multipage_tif_path` | 确认路径正确 | 默认指向 `01 databased/01 Pretest 5/酒精清洗.tif` |
| **24** | `pages` | 选择分析哪些页 | `[1,10,20,30,50,72]` = 6页; `[]` = 全部72页(占3GB) |
| **28** | `pixel_size` | **标定或输入数值** | `'calibrate'` = 交互画线标定; `5e-6` = 直接输入 |
| **29** | `channel_width_um` | 配合 calibrate 使用 | 通道宽度 [um]，来自 CAD: 400 |
| **29** | `diffusion_coeff` | **改成你的DNAPL值** | 分子扩散系数[m^2/s]。TCE → `8.5e-10` |
| **30** | `solubility` | **改成你的DNAPL值** | 水溶解度[mg/L]。TCE → `1100` |
| **31** | `dnapl_density` | **改成你的DNAPL值** | 密度[kg/m^3]。TCE → `1460` |
| **32** | `depth` | **改成你的通道深度** | 通道深度[m]。50um → `50e-6` |
| **37** | `time_vector` | 填入时间戳或留空 | `[]` = 自动生成; `[0,60,120]` = 手动指定 |
| **41** | `rotation_angle` | 自动不准时手动指定 | `[]` = 自动; `-45` = 手动 |
| **45** | `flow_domain_frame` | 通常不用改 | `[]` = 自动用最后一帧 |

**修改示例** — 直接在 config_pipeline.m 第 28-32 行改数值即可：

```matlab
% ---- config_pipeline.m 第 28-32 行 ----
% 改前（默认值）:
    config.physical.pixel_size       = 1e-6;   % [m/pixel]
    config.physical.diffusion_coeff  = 1e-9;   % [m^2/s]
    config.physical.solubility       = 4;      % [mg/L]
    config.physical.dnapl_density    = 1038;   % [kg/m^3]
    config.physical.depth            = 20e-6;  % [m]

% 改后（以 TCE 为例）:
    config.physical.pixel_size       = 5e-6;   % 5 um/pixel
    config.physical.diffusion_coeff  = 8.5e-10;% TCE 扩散系数
    config.physical.solubility       = 1100;   % TCE 溶解度
    config.physical.dnapl_density    = 1460;   % TCE 密度
    config.physical.depth            = 50e-6;  % 50 um 通道
```

#### ADVANCED SETTINGS 区域（第 48 行以下）— 通常不改

| 行号 | 参数 | 默认值 | 什么时候需要改 |
|------|------|--------|---------------|
| **53** | `input_mode` | `'multipage_tif'` | 图像是单独文件时改为 `'separate_files'` |
| **63** | `illumination.kernel_size` | `50` | 光照校正效果不好时调整(30-80) |
| **70** | `segmentation.method` | `'gmm'` | GMM效果不好时改为 `'otsu'` 或 `'adaptive'` |
| **72** | `min_blob_area` | `10` | 噪点太多时增大(30)；小液滴被忽略时减小(5) |
| **73** | `morph_radius` | `2` | 边界毛糙时增大(3)；小液滴被吞掉时减小(1) |
| **82** | `show_figures` | `true` | 批量处理时改为 `false` 节省内存 |

#### 注意事项

**第 18 行路径不要改 `mfilename('fullpath')`**：

```matlab
% 正确 — 自动获取 config_pipeline.m 所在目录:
config.multipage_tif_path = fullfile(fileparts(mfilename('fullpath')), ...
    '01 databased', '01 Pretest 5', '酒精清洗.tif');

% 错误 — mfilename 不接受自定义路径参数:
config.multipage_tif_path = fullfile(fileparts(mfilename('C:\某个路径')), ...
    '01 databased', '01 Pretest 5', '酒精清洗.tif');
```

如果你的 TIF 在别处，直接写绝对路径即可：

```matlab
% 直接写绝对路径（最简单）:
config.multipage_tif_path = 'D:\实验数据\my_experiment.tif';
```

#### 不直接改文件的替代方式

你也可以不改 `config_pipeline.m`，而是在 MATLAB 命令窗口用参数覆盖：

```matlab
% 方式 1: 用 name-value pair 覆盖
config = config_pipeline('physical.pixel_size', 10e-6, ...
                         'physical.dnapl_density', 1460, ...
                         'pages', [1,10,20,30,50,72]);
results = main_DNAPL_pipeline(config);

% 方式 2: 先生成默认配置，再逐个改
config = config_pipeline();
config.pages = [1, 10, 20, 30, 50, 72];
config.physical.pixel_size = 10e-6;
config.physical.dnapl_density = 1460;
config.preprocess.rotation_angle = -45;
results = main_DNAPL_pipeline(config);
```

这种方式不修改源文件，方便对比不同参数的效果。

### 第四步：运行流水线

在 MATLAB 中：

```matlab
% 导航到项目目录
cd 'D:\00 Claude Code\01 MATLAB CODE\01 Microfluid'

% 方式 A：使用默认配置（多页 TIF，全部页面）—— 注意内存！
results = main_DNAPL_pipeline();

% 方式 B（推荐）：只分析部分页面以节省内存
config = config_pipeline('pages', [1, 10, 20, 30, 40, 50, 60, 72]);
results = main_DNAPL_pipeline(config);

% 方式 C：使用自定义参数
config = config_pipeline('physical.pixel_size', 10e-6, ...
                         'segmentation.method', 'otsu');
results = main_DNAPL_pipeline(config);
```

### 第五步：查看结果

运行完成后，结果保存在 `data/output/` 目录：

- `results_*.mat` — 完整结果（MATLAB 格式）
- `blob_properties_*.csv` — 液滴属性表
- `saturation_*.csv` — 饱和度数据
- `timeseries_*.csv` — 时间序列数据
- `*.png` — 可视化图形

## 详细配置说明

> 所有参数的行号和默认值见上方"第三步"表格。本节补充说明各参数的选择依据。

### 输入模式: `input_mode` (第 53 行)

| 值 | 用途 | 需额外配置 |
|----|------|-----------|
| `'multipage_tif'` | 一个 .tif 包含多帧（默认） | 第 18-19 行: `multipage_tif_path`; 第 24 行: `pages` |
| `'separate_files'` | 多个独立文件 | 第 54 行: `image_dir`; 第 55 行: `image_files` |

独立文件模式需要在第 54-55 行填写路径：

```matlab
% 第 54 行:
config.image_dir = 'D:\实验数据\Exp01';
% 第 55 行:
config.image_files = {
    fullfile(config.image_dir, 'frame01.tif')
    fullfile(config.image_dir, 'frame02.tif')
    fullfile(config.image_dir, 'frame03.tif')
};
```

### 页面选择: `pages` (第 24 行)

```matlab
config.pages = [];                        % 读取全部页 (72页, 占~3GB内存)
config.pages = [1, 10, 20, 30, 50, 72];  % 只读6页 (推荐首次使用)
config.pages = 1;                         % 只读第1页 (单帧快速测试)
config.pages = 1:5:72;                    % 每隔5页读一次 (均匀采样)
```

### 流域参考帧: `flow_domain_frame` (第 45 行)

用于提取柱子掩膜的帧。应选 DNAPL 最少的帧（柱子最清晰）。

```matlab
config.flow_domain_frame = [];   % 自动使用最后一帧
config.flow_domain_frame = 72;   % 指定第72页 (多页TIF模式: 页号)
config.flow_domain_frame = 5;    % 指定第5个文件 (独立文件模式: 索引)
```

### Sherwood 关联式: `sherwood_correlation` (第 79 行)

| 值 | 公式 | 适用场景 |
|----|------|---------|
| `'powers'` | Sh = 4.0 * Sn^(-0.6) | 残留 NAPL 溶解 (Powers et al. 1994) |
| `'wilson'` | Sh = 2 + 0.6 * Sn^0.5 | 修正 Wilson-Geankoplis |
| `'interfacial'` | Sh = 2 | 球体在无限介质中 (最简单) |

### 分割方法: `segmentation.method` (第 70 行)

| 值 | 原理 | 优点 | 缺点 |
|----|------|------|------|
| `'gmm'` | Lab a* 通道 + 高斯混合模型 | 最准确，自动找阈值 | 需 Statistics Toolbox |
| `'otsu'` | Lab a* 通道 + Otsu 全局阈值 | 简单快速，无需额外工具箱 | 光照不均匀时效果差 |
| `'adaptive'` | Lab a* 通道 + 局部自适应阈值 | 适应局部光照变化 | 可能过度分割 |

## 常见问题排查

### Q1: 旋转校正不准确

**现象**: 校正后芯片仍然倾斜。

**原因**: 自动检测使用 PCA 分析芯片轮廓主轴方向，对于非矩形芯片或低对比度图像可能失准。

**解决方案**:

```matlab
% 手动指定角度
config = config_pipeline('preprocess.rotation_angle', -45);
% 正值 = 逆时针旋转, 负值 = 顺时针旋转
```

也可以先测试不同角度：

```matlab
config = config_pipeline();
addpath(genpath(pwd));
[images, ~, ~] = read_multipage_tif(config.multipage_tif_path, 1);
rgb = images{1};
for angle = -50:1:-40
    figure; imshow(imrotate(rgb, angle, 'bilinear', 'loose'));
    title(sprintf('角度: %d', angle));
end
```

### Q2: DNAPL 分割效果不好（过多或过少）

**分割过多（噪点）**:

```matlab
config.segmentation.morph_radius = 3;    % 增大（默认2）
config.segmentation.min_blob_area = 30;  % 增大（默认10）
```

**分割过少（遗漏 DNAPL）**:

```matlab
config.segmentation.method = 'otsu';     % 换用 Otsu
% 或使用 adaptive
config.segmentation.method = 'adaptive';
```

**手动检查 a\* 通道**:

```matlab
config = config_pipeline();
addpath(genpath(pwd));
[images, ~, ~] = read_multipage_tif(config.multipage_tif_path, 1);
rgb = images{1};
lab = rgb2lab(im2double(rgb));
figure; imagesc(lab(:,:,2)); colorbar; title('a* 通道');
% 观察 DNAPL 和水的 a* 值范围
```

### Q3: 流域提取不完整

**现象**: 部分柱子未被检测到（尤其是小柱子）。

**解决方案**:

1. 确保 `flow_domain_frame` 指向 DNAPL 最少的帧
2. 检测阈值已优化（MIN_BLOB_AREA=2/3，imopen=1），通常不需要调整
3. 若仍有遗漏，可手动提供流域掩膜：

```matlab
flow_domain = imread('my_flow_domain.tif') > 128;
```

### Q4: 报错 "Undefined function 'fitgmdist'"

**原因**: 缺少 Statistics and Machine Learning Toolbox。

**解决方案**: 安装该工具箱，或切换到 Otsu 方法：

```matlab
config = config_pipeline('segmentation.method', 'otsu');
```

### Q5: 内存不足（多页 TIF）

72 页 RGB TIF 约占 3-4 GB 内存。

**解决方案**:

```matlab
% 方法 1：只分析部分页面（推荐）
config = config_pipeline('pages', [1, 10, 20, 30, 50, 72]);

% 方法 2：关闭图形显示（节省内存）
config.visualization.show_figures = false;

% 方法 3：处理前缩小图像（需修改 read_multipage_tif.m）
```

### Q6: 16-bit TIF 图像

流水线自动处理 16-bit TIF：

- `read_multipage_tif.m` 检测 uint16 并转换为 uint8
- `load_images.m` 同样支持 uint16 → uint8
- 灰度图自动复制为三通道 RGB

无需额外配置。

### Q7: 帧数不匹配 time_vector

```
Warning: time_vector length (5) does not match frame count (8). Auto-generating.
```

**原因**: `config.time_vector` 长度与实际帧数不一致。

**解决方案**: 确保 `time_vector` 长度 = 分析帧数，或留空 `[]` 自动生成。

## 自定义与扩展

### 添加新的分割方法

在 `segmentation/segment_DNAPL.m` 的 `switch` 语句中添加新方法：

```matlab
case 'my_method'
    [is_dnapl, conf, thresh] = my_custom_segmentation(pore_a, pore_b);
```

### 添加新的 Sherwood 关联式

在 `analysis/calculate_mass_transfer.m` 的 `switch` 语句中添加：

```matlab
case 'my_correlation'
    Sh = ...;  % 你的 Sherwood 数公式
```

然后在配置中使用：

```matlab
config = config_pipeline('physical.sherwood_correlation', 'my_correlation');
```

### 添加指进分析（规划中）

`分割方法技术讨论.md` 第 6 章详细讨论了两个待实现的分析模块：

1. **指进分析 `analyze_fingering.m`** — 骨架提取 + 拓扑指标（指头数、分支密度、流型自动分类）
2. **分形维数 `box_counting_dimension.m`** — 盒计数法量化 DNAPL 形态复杂度

这两个模块输入为 `dnapl_mask`（已有），不影响现有代码。详见技术讨论文档。
